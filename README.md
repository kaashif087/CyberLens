# CyberLens – Cybersecurity Data Analysis using Python

## Overview

CyberLens is a cybersecurity data analysis project developed using Python and Jupyter Notebook to analyze cybercrime incident data. The project applies data preprocessing, exploratory data analysis (EDA), and visualization techniques to uncover trends, patterns, and insights from cybersecurity datasets.

The primary objective is to transform raw cybercrime data into meaningful information that supports better understanding of security trends and data-driven decision-making.

---

## Objectives

* Analyze cybersecurity incident datasets.
* Clean and preprocess raw data.
* Perform Exploratory Data Analysis (EDA).
* Visualize trends and patterns in cybersecurity incidents.
* Generate actionable insights from the dataset.

---

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/kaashif087/CyberLens.git
```

2. Navigate to the project directory.

```bash
cd CyberLens
```

3. Install the required Python libraries.

```bash
pip install -r requirements.txt
```

4. Open the Jupyter Notebook and run all cells.
